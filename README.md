# CYBER_Project
MultiModal Behavioral Observation during Psychometric Testing using IoT & AI" ( Erasmus Mundus CYBER, Université Paris Cité / Centre Borelli ).
The pipeline combines a Moofit HW401 heart rate sensor, mouse dynamics, and keyboard interaction logging (Thales platform) with the BFI-2-S personality instrument and NAPS affective image stimuli rated via Russell's Circumplex Model and Plutchik's Wheel of Emotions.
Note: Use of synthetic dataset (N=20, 10F/10M, seed=42) generated from literature-informed effect sizes (Morris, White & Crowther, 2019) for first-order pipeline verification only.
